vrme_server::accounts

[−][src]Module vrme_server::accounts::register

Handles user registration and account creation.

We require that the client supply a hashed password from the client-side which shall be generated by a cryptographically-strong hashing function such as SHA-256 to avoid transferring the plain-text password over the network which could be compromised.
